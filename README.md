# MintynApp
An App to demonstrate certain features and functionalities in iOS App using MyMintApp as  reference.
MyMintApp is a Fintech Application that offers numerous financial services to users.


# Requirements
- iOS ~> 13
- Xcode ~> 13 (13 compatible)
- Swift ~> 4.2

# Installation
Clone the repository
```sh
$ git clone https://github.com/OBaller/MintynApp.git
$ cd MintynApp
$ pod install
```
# Support
- Dark Mode Support

Open the file `MintynApp.xcworkspace` using Xcode
Allow the project to build and update build settings.
Click on the play button at the top left corner of XCode to build and run the project
